Dokumen Spesifikasi Aplikasi Mobile
1. Informasi Umum
Nama Aplikasi: CuanKlik
Deskripsi Singkat:
Kami akan merancang sebuah aplikasi yang memiliki fitur kalkulator penghitung Harga Pokok Penjualan (HPP). Cara kerja aplikasi ini adalah dengan memasukkan angka Biaya Tetap, Biaya Variabel per produk serta Target Margin Keuntungan, kemudian aplikasi akan mengkalkulasikan analisis profit yaitu prediksi produk yang harus dijual agar memperoleh keuntungan.
Target Pengguna: Pelaku Usaha Mikro Kecil dan Menengah (UMKM)
